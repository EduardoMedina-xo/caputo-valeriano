Installation

SR Ebook currently is not included in the SR App autodeployment.

Usage

1. Adding a new chapter

The number of chapters displayed is controlled by a choice field labeled Number of Chapter.
The maximum number of chapters supported is 20.

1. Updating sidebar nav

Each sidebar nav item is created using a text module. Each has an automatically-assigned unique ID that links to each ebook chapter.
